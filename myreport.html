AlgoSeek Monthly Options Data Summary

  • TotalTrades : sum of daily trades over the month.
  • MinTrades : number of trades on day with the minimum number of trades in the month.
  • MaxTrades : number of trades on day with the maximum number of trades in the month.
  • Empties : number of days with no trades.
  • Under50 : number of days with trades < 50.
  • Under100 : number of days with 50 < trades < 100.
  • Under500 : number of days with 100 < trades < 500.
  • Under1000 : number of days with 500 < trades < 1000.
  • Over1000 : number of days with trades > 1000.

    AlgoSeek Daily Options Data Summary
